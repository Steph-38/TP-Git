Bonjour tout le monde !

Pour ce TP sur Git, jai réutilisé le travail effectué pour un autre TP openclassrooms.
Mon projet est donc une carte d'anniversaire animée en JQuery.
Au lancement du fichier index.html dans le navigateur, un message Joyeux anniversaire apparait.
Si on clique sur le bouton Cliquez-moi, un second message de voeux apparait.
Si on clique sur le second message, un troisième message de voeux apparait.
Pareil si on clique sur le troisième message ...
Enfin, si on clique sur le quatrième et dernier message, un "Joyeux anniversaire" remplace les messages de voeux.

Mon projet va donc se composer de 4 fichiers déjà créés :
1 - Un fichier index.html pour la base du code de la carte d'anniversaire
2 - Un fichier styles.css pour la mise en forme
3 - Le fichier jquery-3.3.1.js indispensable pour l'utilisation du framework JQuery
4 - Un fichier anniversaire.js qui contiendra le script pour animer la carte

Pour coller au plus près du TP git et afin de ne pas trop charger le projet,
je vais modifier les fichiers index.html, styles.css, et anniversaire.js.
Je vais simplifier le projet de base en supprimant notamment les animations sonores et celles avec des images.

Merci d'avance pour la correction !
